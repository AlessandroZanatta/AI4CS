# Artificial Intelligence for CyberSecurity

Project for the course of Artificial Intelligence for CyberSecurity. This consists in the analysis of a dataset related to CyberSecurity using data mining/machine learning techniques.

The chosen dataset, available in the [dataset](src/dataset) folder, is taken from [this website](https://www.unb.ca/cic/datasets/malmem-2022.html). It is a dataset regarding malware detection using features from memory dumps.
